# **App Name**: DomicileEase

## Core Features:

- Form Input: Allows users to input required information for domicile certificate application.
- Photo Upload: Enables users to upload a photo for the application; supports progress tracking and validation.
- PDF Generation: Generates a PDF document based on the user input and photo, formatted according to the specified template.
- Error Handling: Generates comprehensive error messages.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and officialdom.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Subtle teal (#4DB6AC) for interactive elements and highlights.
- Headline font: 'Literata', serif, to communicate a literary, slightly formal feel
- Body font: 'Inter', sans-serif, providing a modern, machined, objective, neutral feel.
- Use government-themed icons where appropriate to enhance visual understanding.
- Maintain a structured, form-based layout for easy data entry and review.
